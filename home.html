ABOUT

LEGACY

It is difficult to keep it short; I've been around for some time — been there, done that. But there is a lot more to my story. I am a second-generation computer programmer — it is equivalent to saying I was raised by wolves. Growing up surrounded by my father and colleagues-friends talking about Assembly, Cobol, Fortran, JCL, and writing complex programs to run on 8K RAM only. Logic efficiency, resources austerity, no time to waste processing jobs — it was a must. I've been thru that also, developing software for very limited micro-computer. The bottom line is, I still carry the same professional discipline today, I always deliver my best effort on all my projects.
